Project Report

Your Name: Geonwoo

Introduction

This portfolio represents a future version of myself as a Senior Project Manager in the construction industry. It showcases my skills, achievements, and key projects that I have worked on, such as the Seoul Mega Tower and Incheon Bridge Expansion. The target audience includes potential employers, collaborators, and clients who are looking for an experienced professional in the field of construction project management.

Inspiration

  • Studio Bagaz: I was inspired by the neat portfolio. I loved the contact page of this portfolio and tried to have similar contact page as it.
  • Align Wen Photography: This portfolio used different backgrounds for every page and I tried to do the same thing for my portfolio pages. However, I felt a bit messy when the backgrounds are different for every page, so I just put the same background for every page.
  • Rafael Varona: The portfolio navigates me to about page when I press 'read more' so I added the same function to my portfolio.

Accessibility

  • All images include descriptive alt text to support screen readers.
  • High-contrast color combinations ensure readability for users with visual impairments.
  • The website is fully keyboard navigable, allowing users with motor impairments to browse the site.

Usability

  • The navigation menu is simple and consistent across all pages for ease of use.
  • Responsive design ensures the site looks good and functions properly on all devices, including desktops, tablets, and smartphones.
  • Clear headings and well-structured content make it easy for users to find the information they need.

Learning

  • I learned how to use CSS Grid and Flexbox to create a responsive layout for the project showcase.
  • Working with JavaScript to implement interactive features such as image sliders was new to me, and I acquired this skill through online tutorials.
  • I improved my understanding of accessibility standards, ensuring that the website is usable for people with different abilities.

Evaluation

The most successful aspect of my portfolio is its clean, professional design, which effectively showcases my work in a way that is easy for users to navigate. I am particularly proud of the responsive design and how the portfolio adapts to different screen sizes. However, I would like to improve the performance of the website by optimizing the loading time for images and further enhancing the interactivity with JavaScript features.

Resources

  • Google Fonts for typography (Roboto, Arial).
  • FontAwesome icons for navigation.
  • Image resources from Unsplash for placeholder project images.
  • CSS Grid and Flexbox tutorials from MDN and W3Schools.
